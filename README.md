# nlp-parser-chatbot

Lightweight Vietnamese EV consulting chatbot using a custom NLP parser, FastAPI, and a small editable vehicle database.

## Run

```bash
pip install -r requirements.txt
python run.py
```

Open `http://localhost:8000`.

## Supported examples

- `VF3 giá bao nhiêu?`
- `Xe điện nào dưới 500 triệu chạy xa nhất?`
- `Tư vấn xe dưới 700 triệu phù hợp nhất`
- `So sánh VF5 và VF6`
- `Xe nào đi được trên 400 km?`
